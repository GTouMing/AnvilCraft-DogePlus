**AnvilCraft Doge+** 为 [AnvilCraft](https://github.com/Anvil-Dev/AnvilCraft) 添加了四种溜槽变体与磁化节点物品捕获增强。

---

## 内容

### 方块

| 方块 | 合成 | 功能 |
|---|---|---|
| 溜槽发射器 | 溜槽 + 发射器 | 从上方吸入物品并向 facing 方向发射 |
| 溜槽投掷器 | 溜槽 + 投掷器 | 从上方吸入物品并向 facing 方向投出 |
| 磁力溜槽发射器 | 磁力溜槽 + 发射器 | ↑ 支持朝上放置，输入来自输出反向 |
| 磁力溜槽投掷器 | 磁力溜槽 + 投掷器 | ↑ 支持朝上放置，输入来自输出反向 |

所有方块均支持物品过滤、槽位禁用、比较器输出。

### 增强：磁化节点物品捕获

磁化节点（`anvilcraft:magnetized_node`）生成时会自动创建节点支撑实体，将周围掉落物吸入并环绕展示（最多 8 组，自动合并同类物品）。空手右键取回全部，手持物品右键存入。节点消失时自动释放所有物品。
